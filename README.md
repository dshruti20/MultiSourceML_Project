# MultiSourceML_Project
Instructions to run this project are as below,<br />
Clone this Repository,<br />
git clone https://github.com/dshruti20/MultiSourceML_Project.git <br />
Run requirement.txt file to install the dependencies. It is important to have specified versions to run the files successfully. <br />
pip install -r requirements.txt <br />
To get the Memory Prediction and Job Completion Time for input model, run below command, <br />
python maincode.py -m resnet50 -b 64 -o inference -g 3080 <br />
